# ClineIAI Playground

React/TypeScript Projekt verbunden mit:
- **StackBlitz** (Live-Preview im Browser)
- **VS Code + Cline** (KI-gestütztes Debugging)
- **GitHub** (Synchronisation)

## Workflow

1. Code in VS Code bearbeiten
2. Cline für KI-Unterstützung nutzen
3. `git push` → StackBlitz aktualisiert automatisch
4. Live-Preview im Browser testen

## Befehle

```bash
npm install    # Abhängigkeiten installieren
npm run dev    # Entwicklungsserver starten
npm run build  # Für Produktion bauen
```
